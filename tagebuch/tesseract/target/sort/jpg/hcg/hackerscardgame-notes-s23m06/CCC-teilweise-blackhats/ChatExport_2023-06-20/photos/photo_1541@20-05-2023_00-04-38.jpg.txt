Open the folder where you have installed Tor,
browse through browser > Tor browser > Tor >
data. There you will find a file named "torre."
Open the file with Notepad. At the end of the
document, on a new line, write this: ExitNodes
{US}. For example, if you want your exit node
to be in China, type {CN}. 073.2023
