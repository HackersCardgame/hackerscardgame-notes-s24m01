Das Beobachter-Muster (englisch observer
pattern, auch listener pattern) ist ein
Entwurfsmuster aus dem Bereich der
Softwareentwicklung. Es gehort zur
Kategorie der Verhaltensmuster (englisch
behavioural patterns) In der Programmier-
sprache Java ware es public interface
Observer und verwendet wird es mit
“implements Observer" (braucht Ubung)

Anwendungsfalle

dient der Weitergabe von Anderungen an Objekt
zu von diesem Objekt abhangige Strukturen
