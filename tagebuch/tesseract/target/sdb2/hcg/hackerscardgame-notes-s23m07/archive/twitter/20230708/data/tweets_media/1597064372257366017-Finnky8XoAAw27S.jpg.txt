Schliissel-Werte-Datenbank

Key Value Database oder Key Value

Store) dient zur elektronischen

Datenverwaltung in Computersystemen. Sie
basiert auf dem Schlissel-Werte-Datenmodell,
um assoziative Datenfelder zu speichern.

Die zugehorige Software ist das Schlissel-
Werte-Datenbankmanagementsystem.
