Das Beobachter-Muster (englisch observer
pattern / listener pattern) ist ein
Software Entwurfsmuster. Es gehort zur
Kategorie der Verhaltensmuster (englisch
behavioural patterns). In der Programmier-
sprache Java ware es public interface
Observer und verwendet wird es mit
“implements Observer" (Verwenduung braucht
ein wenig Ubung & einen guten Tutor)

Anwendungsfalle
dient der Weitergabe von Anderungen an
Objekte die von diesem Objekt abhdangen.
